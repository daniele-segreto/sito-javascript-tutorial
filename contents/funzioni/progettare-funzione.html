Progettare una funzione

Le funzioni sono uno degli elementi costitutivi fondamentali di JavaScript. Una funzione in JavaScript è simile a una procedura: un insieme di istruzioni che esegue un'attività o calcola un valore, ma affinché una procedura possa qualificarsi come una funzione, dovrebbe ricevere un input e restituire un output in cui esiste una relazione ovvia tra ingresso e uscita. Per utilizzare una funzione, devi definirla da qualche parte nell'ambito da cui desideri chiamarla. Vedi anche il capitolo di riferimento esaustivo sulle funzioni JavaScript per conoscere i dettagli.

Dichiarazioni di funzioni

Una definizione di funzione (chiamata anche dichiarazione di funzione o istruzione di funzione ) è costituita da functionparola chiave, seguita da:

- Il nome della funzione.

- Un elenco di parametri della funzione, racchiusi tra parentesi e separati da virgole.

- Le istruzioni JavaScript che definiscono la funzione, racchiuse tra parentesi graffe, { /* … */ }.

Ad esempio, il codice seguente definisce una semplice funzione denominata square:

    
      function square(number) {
        return number * number;
      }
