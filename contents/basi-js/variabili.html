Variabili

Istruzioni

L'istruzione var dichiara variabili con ambito funzione o con ambito globale, facoltativamente inizializzandole ciascuna su un valore.

Esempio: 

  
    var x = 1;
    
    if (x === 1) {
      var x = 2;
      
      console.log(x);
      // Expected output: 2
    }
    
    console.log(x);
    // Expected output: 2
