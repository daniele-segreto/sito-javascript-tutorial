Operatore di confronto (> < != ==)

Un operatore di confronto confronta i suoi operandi e restituisce un valore logico a seconda che il confronto sia vero. numerici, stringa, logici o oggetto Gli operandi possono essere valori . Le stringhe vengono confrontate in base all'ordinamento lessicografico standard, utilizzando valori Unicode. Nella maggior parte dei casi, se i due operandi non sono dello stesso tipo, JavaScript tenta di convertirli in un tipo appropriato per il confronto. Questo comportamento generalmente comporta il confronto numerico degli operandi. Le uniche eccezioni alla conversione dei tipi all'interno dei confronti riguardano il file ===E !==operatori, che eseguono confronti rigorosi di uguaglianza e disuguaglianza. Questi operatori non tentano di convertire gli operandi in tipi compatibili prima di verificare l'uguaglianza. La tabella seguente descrive gli operatori di confronto in termini di questo codice di esempio:

Esempio: 

  
    a = 10;
    b = 15;

    // verifica se 'a' è maggiore di 'b'
    a > b

    // verifica se 'a' è minore di 'b'
    a < b

    // verifica se 'a' è uguale a 'b'
    a == b
    
    // verifica se 'a' è diverso da 'b'
    a != b
