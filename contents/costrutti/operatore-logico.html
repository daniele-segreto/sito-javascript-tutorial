Operatore logico (&& ||)

Gli operatori logici vengono generalmente utilizzati con valori booleani (logici); quando lo sono, restituiscono un valore booleano. comunque, il &&E ||gli operatori effettivamente restituiscono il valore di uno degli operandi specificati, quindi se questi Gli operatori vengono utilizzati con valori non booleani, possono restituire un valore non booleano. Gli operatori logici sono descritti nella tabella seguente.

Esempio: 

  
    // && ||
    a = 10;
    b = 15;
    c = 15;

    // verifica che entrambe le condizioni siano vere
    a < b && b = c;

    // verifica che almeno una delle due condizioni sia vera
    a > b || b = c;
