Operatore condizionale (if/else)

L'istruzione if / else esegue un'istruzione se una condizione specificata è vera . Se la condizione è false , un'altra istruzione nell'opzionale elsela clausola verrà eseguita.

Sintassi: 

  
    if (condition)
      statement1
    
    // With an else clause
    if (condition)
      statement1
    else
      statement2

condition: rappresenta un'espressione considerata vera o falsa.

statement1: istruzione che viene eseguita se la condizione è vera . Può essere qualsiasi istruzione, comprese quelle ulteriormente nidificate ifdichiarazioni. Per eseguire più istruzioni, utilizzare un'istruzione di blocco ( { /* ... */ }) per raggruppare tali affermazioni. Per non eseguire alcuna istruzione, utilizzare un'istruzione vuota .

statement2: rappresenta un'espressione considerata vera o falsa.
