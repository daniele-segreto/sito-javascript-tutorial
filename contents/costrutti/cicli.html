Cicli (for/while)

for statement

Un ciclo for si ripete finché una condizione specificata non risulta falsa. Il JavaScript foril ciclo è simile a Java e C forciclo continuo. In un ciclo for'istruzione appare come segue:

Esempio: 

  
    for (initialization; condition; afterthought)
    statement

Quando un forviene eseguito il ciclo, si verifica quanto segue:

1. L'espressione di inizializzazione initialization, se presente, viene eseguito. Questa espressione solitamente inizializza uno o più contatori di ciclo, ma la sintassi consente un'espressione di qualsiasi grado di complessità. Questa espressione può anche dichiarare variabili.

2. Il conditionviene valutata l'espressione Se il valore di conditionè vero, le istruzioni del ciclo vengono eseguite. Altrimenti, il foril ciclo termina. (Se la conditionl' espressione viene omessa completamente, si presuppone che la condizione sia vera.)

3. Lo statemente segue. Per eseguire più istruzioni, utilizzare un'istruzione di blocco ( { }) per raggruppare tali affermazioni.

4. Se presente, l'espressione di aggiornamento afterthoughtviene eseguito.

5. Il controllo ritorna al passaggio 2.
