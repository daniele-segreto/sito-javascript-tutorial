Gestire gli array

Notazione letterale degli array

Gli array possono essere creati utilizzando il valore letterale notazione:

    
        const fruits = ["Apple", "Banana"];

        console.log(fruits.length); // 2
        console.log(fruits[0]); // "Apple"

Costruttore di Array con un singolo parametro

Gli array possono essere creati utilizzando un costruttore con un singolo parametro numerico. Viene creato un array con suo lengthproprietà impostata su quel numero e gli elementi dell'array sono vuoti slot. 

    
        const arrayEmpty = new Array(2);

        console.log(arrayEmpty.length); // 2
        console.log(arrayEmpty[0]); // undefined; actually, it is an empty slot
        console.log(0 in arrayEmpty); // false
        console.log(1 in arrayEmpty); // false
    

    
        const arrayOfOne = new Array("2"); // Not the number 2 but the string "2"

        console.log(arrayOfOne.length); // 1
        console.log(arrayOfOne[0]); // "2"

Costruttore di array con più parametri

Se al costruttore viene passato più di un argomento, un nuovo Arraycon vengono creati gli elementi specificati. 

    
        const fruits = new Array("Apple", "Banana");

        console.log(fruits.length); // 2
        console.log(fruits[0]); // "Apple"
