Metodi di manipolazione degli array

Push

Usiamo .push quando vogliamo aggiungere qualcosa alla fine di un array. Questo metodo ci consente di aggiungere uno o più elementi alla fine di un array. Il metodo push restituisce la nuova lunghezza dell'array.

Aggiunta di un elemento alla fine di un array 

    
        let numbers = [1,2,3,4]
        numbers.push(5)
        
        console.log(numbers) // [1,2,3,4,5]

Pop

Usiamo .pop quando vogliamo rimuovere qualcosa dalla fine di un array. Il metodo .pop restituisce l'elemento che è stato rimosso . Se l'array è vuoto, restituirà undefinito.

    
        let fruits = ["🍏", "🍊", "🍌"]

        fruits.pop() // "🍌"
        console.log(fruits) // ["🍏", "🍊"]
        
        fruits.pop() // "🍊"
        console.log(fruits) // ["🍏"]
        
        fruits.pop() // "🍏"
        console.log(fruits) // []
        
        fruits.pop() // undefined
        console.log(fruits) // []

Unshift

Usiamo .unshift quando vogliamo aggiungere qualcosa all'inizio di un array. Questo metodo ci consente di aggiungere uno o più elementi all'inizio di un array. Il metodo .unshift restituisce la nuova lunghezza dell'array. 

    
        let numbers = [2,3,4]
        numbers.unshift(1)
        
        console.log(numbers) // [1,2,3,4]

Shift

Usiamo .shift quando vogliamo rimuovere qualcosa dall'inizio di un array. Il metodo .shift restituisce l'elemento che è stato rimosso . Se l'array è vuoto, restituirà undefinito. 

    
        let fruits = ["🍏", "🍊", "🍌"]

        fruits.shift() // "🍏"
        console.log(fruits) // ["🍊", "🍌"]
        
        fruits.shift() // "🍊"
        console.log(fruits) // ["🍌"]
        
        fruits.shift() // "🍌"
        console.log(fruits) // []
        
        fruits.shift() // undefined
        console.log(fruits) // []
