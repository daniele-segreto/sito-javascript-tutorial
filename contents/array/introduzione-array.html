Introduzione agli array

Array

In JavaScript, gli array non sono primitivi ma lo sono Arrayoggetti con le seguenti caratteristiche principali:

- Gli array JavaScript sono ridimensionabili e possono contenere un mix di diversi tipi di dati. (Quando tali caratteristiche sono indesiderabili, utilizzare invece array tipizzati.)

- Gli array JavaScript non sono array associativi e quindi non è possibile accedere agli elementi dell'array utilizzando stringhe arbitrarie come indici, ma è necessario accedervi utilizzando numeri interi non negativi (o la rispettiva forma di stringa) come indici.

- Gli array JavaScript sono indicizzati a zero : il primo elemento di un array è in indice 0, il secondo è all'indice 1, e così via, e l'ultimo elemento ha il valore dell'array lengthmeno proprietà 1.

- JavaScript Le operazioni di copia di array creano copie superficiali . (Tutte le operazioni di copia integrate standard con qualsiasi oggetto JavaScript creano copie superficiali, anziché copie profonde ).
